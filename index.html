How to use the calculator

  • Digits: 0–9
  • Decimal point: .
  • Operators: + - * /
  • Calculate: Enter
  • Delete digit: Backspace
  • Clear: Esc (C)
  • Extras: ±, %, x², √x, 1/x, Memory (MC/MR/M+/M-)
  • Divide by zero: shows “Error”.
